Object Example

The following class file would define an object (Product) to be exposed/manipulated by the web service. 


    <?php
    class Product extends AbstractObject{

        protected $_table = 'db_product_table';
        protected $_primary = 'db_primary_key_id';
        protected $_describe = true; // get fields from DB
        protected $_cache = true; // cache reads

        protected function afterRead(){
            $this->name = strtolower($this->name);
            $this->extra_field = 'EXTRA';
        }

        protected function afterEnumerate(&$results){
            foreach($results as $result){
                $result->name = strtolower($result->name);
                $result->extra_field = 'EXTRA';
            }
        }
    }
    ?>
