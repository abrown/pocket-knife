JSON Service Example

The following code would define a service endpoint at the file app.php. Requests for objects (e.g. http://www.example.com/app.php/messages/), etc., would return JSON responses. 


    <?php
    require 'knife/start.php';
    $service = new KHtml();
    $service->setAnchor('app.php');
    $service->allow('messages');
    $service->handle();
