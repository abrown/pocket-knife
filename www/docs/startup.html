PocketKnife: A PHP Tool for Web Development

What happens on including 'knife/start.php'

The following are created (in order):

NameTypeDescriptionMemory Usage
KNIFE_BASE_PATH Constant Absolute path to current Knife folder
DS Constant Shortcut for OS directory separator
pr Function Prints out debug information
get_public_vars Function Gets public variables of a class instance
__autoload Function Overrides PHP __autoload to load classes from Knife files
require KSettings Class Loads Settings, cached
KNIFE_DEBUG_ON Constant KSettings sets debugging level, including ini_set('display_errors', x) and error_reporting()
