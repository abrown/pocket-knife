PocketKnife: A PHP Tool for Web Development

  • PHP5

  • _autoload

Definition

PocketKnife is a mini-framework designed to quickly setup simple web services. The framework is very simple, so it may need to be modified for production use; however, it does provide the developer with a solid platform to begin building.

Requirements

The web service uses the following components, for which you may need to have a recent version of PHP:

  • PHP5 syntax (__construct, public, private, etc.)
  • PDO
  • __autoload function (may need to rewrite this if the service is run within an application using it already)
  • .htaccess rewrites (to set Digest Authentication variables in $_SERVER)
  • mysql
  • transactions (with InnoDB)

Setup

To set up the web service:

  1. Copy the files to the right directories
  2. Configure the /config/main.php file; you may add any additional configuration variables in other files within this folder. Also, you may want to make sure the .htaccess files are configured like you want.
  3. Create data objects as extensions of the AbstractObject class and put the path to these classes in the configuration file under $class_paths. You may choose to not extend AbstractObject: if this is the case, note that AbstractAction expects at least these methods: __bind, __getID, and __setID. If you do extend AbstractObject, see its class file for a description of available options.
  4. If extending AbstractObject, overload functions (e.g. beforeRead) to change their logic. Also, see the AbstractObject class file for a description of available options.
